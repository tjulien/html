html.tool
===========

inspired by json.tool, a pretty printer for html
	 $ echo '&lt;html&gt;&lt;body&gt;Hello World&lt;/body&gt;&lt;/html&gt;' | python -mhtml.tool
	 &lt;html&gt;
 	  &lt;body&gt;
  	   Hello World
 	  &lt;/body&gt;
	 &lt;/html&gt;
